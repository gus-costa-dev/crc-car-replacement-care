## 📋 About The Project

C.R.C. (Car Replacement Care) is an internal staff management system for an accident replacement vehicle company.

When a customer is involved in a not-at-fault accident, they submit a claim through the public form. Staff then manage the claim through the internal dashboard — assigning replacement vehicles, updating claim status, and liaising with insurers.

---

## ✨ Features

### Public Facing
- 📋 Customer accident claim submission form
- ✅ Real-time form validation
- 🔄 AJAX submission — no page reloads
- 📱 Responsive design

### Staff System
- 🔐 Secure authentication (register/login/logout)
- 📊 Dashboard with live stats
- 🗂️ Full Claims CRUD (Create, Read, Update, Delete)
- 🚙 Full Vehicle Fleet CRUD
- 🔍 Claims list with vehicle assignment via LEFT JOIN
- 🏷️ Status management (Pending, Approved, Rejected, Completed)
